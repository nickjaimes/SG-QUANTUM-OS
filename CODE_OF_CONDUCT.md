# Code of Conduct

## Our Pledge
We aim to foster an open, welcoming, diverse environment  
for all contributors, users, and collaborators.

## Standards
✅ Use respectful language  
✅ Be collaborative  
✅ Offer constructive feedback  
✅ Respect differing viewpoints  

❌ No harassment  
❌ No insults or discrimination  
❌ No inappropriate content  

## Enforcement
Report issues:  
security@safewayguardian.org (placeholder)  
or GitHub Issues

Violations may result in warnings or removal from participation.
