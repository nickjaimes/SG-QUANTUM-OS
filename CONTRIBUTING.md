# Contributing to SG QUANTUM OS v2.0

Thank you for your interest in contributing!  
SG-QOS is co-created by Nicolas E. Santiago and enhanced with AI-assisted design using ChatGPT.

We welcome:
✅ Bug reports  
✅ Feature requests  
✅ Documentation improvements  
✅ Code contributions  

---

## How to Contribute

### 1) Fork + Clone
\`\`\`bash
git clone https://github.com/<your-fork>/SG-QUANTUM-OS-v2.0.git
cd SG-QUANTUM-OS-v2.0
\`\`\`

### 2) Create a Feature Branch
\`\`\`bash
git checkout -b feat/<name>
\`\`\`

### 3) Make Changes
Follow project structure under:
\`/core\`, \`/docs\`, \`/tools\`

### 4) Commit Messages
Use conventional commits format:
\`\`\`
feat: add TRINITY routing improvement
fix: correct HIMS anomaly parser
docs: update INSTALL_GUIDE
\`\`\`

### 5) Pull Request
- Describe your change
- Reference related issues
- Ensure no large binaries (>100MB)
- PR must pass automated checks

---

## Code Style
- Clear naming
- Modular architecture
- Comment critical logic
- Document new components
