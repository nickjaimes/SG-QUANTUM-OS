# Security Policy

## Supported Versions
Actively secured: v2.x

## Reporting a Vulnerability
If you discover a security issue, please:
1) Do NOT disclose publicly
2) Contact maintainers:
   security@safewayguardian.org  (placeholder)
3) Provide:
   - Details
   - Reproduction steps
   - Impact assessment

## Responsible Disclosure
We will:
- Validate report
- Provide guidance
- Patch in a timely manner
- Credit the reporter (optional)

Thank you for helping keep SG QUANTUM OS secure.
