# Developer Setup Guide — SG QUANTUM OS v2.0

## Overview
This guide helps developers set up, develop, and test components  
of SG-QOS locally or via VM.

---

## 1) Requirements
- Linux/macOS
- Python 3.10+
- Git
- Node.js (for UI components)
- Recommended: VSCode
